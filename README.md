# FlexArray

An implementation of immutable functional arrays using Braun trees.
See: "ML for the working programmer".

### pre-reqs:

requirements are:
```
OCaml
Oasis (0.3+)
oUnit (for tests)
core (to run sample)
```

### Installing

```
oasis setup
ocaml setup.ml -all
ocaml setup.ml -install # may require sudo
```

### Usage

See `lib/flexArray.mli`
